Class: VDataReader

VDataReader(readers, signalsopt, insensitiveCaseopt, queryFilteropt, queryStartTimeopt, queryEndTimeopt, readLivingDataopt, columnExpandModeopt, signalQueueModeopt)

A class to read to read and decode data according to format specification. Examples: let reader = factory.open(); let frame = reader.df();
